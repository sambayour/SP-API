# SP-API

### Installation

```bash
$ npm ci
```

### Database

A database sample file(.sql) has been included in the root file.

### Running the app

Duplicate the env.exmaple and rename it to .env, update all the variables.

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
